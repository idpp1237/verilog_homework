黃燈 顯示關卡的狀態(5關) 每一關敵方的血量都會不同                                                                                                                                       
紅燈 遊戲結束(被擊殺)

Fire2 在時間內按下越多次，給予敵方的傷害越高                                                                                                                                           
RESET 遊戲重新開始                                                                                                                                                                   

銀幕上右邊的是玩家 左邊是敵方 上面是他們的血條
